# Xenon-Book-Analyser
Offline AI-powered book analyser that can summarize books, generate tests,  and convert content into MP3 using LLaMA3 (GPU required).
#Works faster in GPU devices.
Steps to Use:
1.  Open the terminal and Install all require Modules. code is given below.
   pip install PyPDF2 pyttsx3
2.  Install the Offline AI Llama3
   ollama pull llama3
3. Copy your PDF of Books in Books Folder.
4. run book_analyser.py file.
